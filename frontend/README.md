# Balruno — Frontend

게임 스튜디오 워크스페이스 (밸런싱 + 애자일 PM + 로드맵) 의 Next.js 16 프론트엔드.

메인 프로젝트 문서는 [루트 README](../README.md) 참조.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
